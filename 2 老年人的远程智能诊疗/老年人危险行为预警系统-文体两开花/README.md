#老年人危险行为预警系统

#应用场景介绍
老年人跌倒事件时常发生，老年人即使受到很小的伤害，也可能会对他的生活和健康造成很大影响。危险行为预测系统通过手机陀螺仪分类算法，提前判断老年人危险行为事件，如跌倒，走路颠簸，摇摇晃晃等行为的提前预警。
